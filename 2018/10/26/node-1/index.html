Blog 服务端代码

中间件：就好比我们浏览器的插件，插件可以弥补浏览器某些方面的不足，而中间件相对于Express而言与之相同。

body-parser是一个HTTP请求体解析中间件，使用这个模块可以解析JSON、Raw、文本、URL-encoded格式的请求体，Express框架中就是使用这个模块做为请求体解析中间件。

mongodb 配置流程

mongodb 安装过程不必多说，进度条加载完毕即可。

配置:
打开安装mongodb的硬盘，在根目录下新建“data”文件夹，进入到“data”文件夹，新建’db’文件夹。=> 找到 mongodb 安装位置，并进入到 mongodb 文件夹，双击 mongod.exe. 在出现的命令行里面找 success 找到则证明配置成功！

若希望通过服务器连接到数据库，就要把mongod.exe打开，才能连接上。如果动手能力强，可以试着配置mongdb，使它可以跟随 windows 的启动而启动。
