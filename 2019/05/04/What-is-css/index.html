什么是 CSS

CSS 是用于指定文档(HTML)如何呈现的语言

CSS 是如何影响HTML？

web浏览器将“css规则”应用于文档以影响它们的显示方式。

什么是css规则呢？

一个声明块：如何显示HTML(属性名、属性值)
一个选择器：选择元素，这些元素是想要应用上述属性的HTML

什么是声明块？

一个css声明是由属性名和对应的属性值组成(决定了HTML如何显示)，而多个声明组成了声明块。

什么是选择器?

既然我们已经声明了多个css声明，那么这些声明应用在哪些HTML标签上呢？

通过在每个声明块前加上选择器把声明块与HTML标签连接起来。
